## CG集ビジュアルの鉄則

### CG集特有の構図テクニック

1. **サンプル映え**: 3ページ目と7ページ目はサンプル画像に使われやすい。構図とエロ度を特に意識
2. **顔と行為の両立**: 顔が見える構図を7割以上にする。表情が見えないとCG集の価値が半減
3. **差分意識**: 同じ体位でもアングルを変えるだけで差分感が出る
4. **背景の変化**: 場所が同じでも照明（明→暗）や散らかり具合で変化を出す
5. **ズーム緩急**: 寄り→引き→寄り のリズムで飽きさせない

### 背景・場所の多様性ガイド

**背景変化の必須ルール:**
- 3シーン連続で同じ背景locationを使わない（最重要）
- 全シーンの50%以上が同一背景base tagにならないこと
- 10シーン中最低2つの異なる背景を使う

**同一場所でも変化をつける方法:**
- 照明の変化: natural_lighting → dim_lighting → candlelight
- 時間帯の変化: morning → afternoon → night
- 室内状態の変化: tidy → messy → clothes_on_floor
- アングルで部屋の違う面を見せる: window側 → bed側 → door側
- 季節小物の変化: 窓の外の景色、室温を示すディテール

**場所遷移の推奨パターン（10ページの場合）:**
1-2. 導入: 教室/オフィス等の公共空間
3. 移動: 廊下/階段/車内（場面転換の緩衝）
4-9. 本番: 寝室/ホテル（途中で照明・状態変化をつける）
10. 余韻: 同じ場所だが朝の光/散らかった状態

**背景タグの入れ方:**
背景タグはsd_promptの**行為タグの後、照明タグの前**に入れる。
例: `..., sex, vaginal, bedroom, bed, pillow, dim_lighting, ...`

### sd_promptでの構図タグの入れ方
構図タグは**キャラタグの直後、行為タグの前**に入れる。
例: `(masterpiece, best_quality:1.2), キャラタグ, from_above, pov, sex, vaginal, ...`

## 8. ショット配分メトリクス（新規）

### 10ページCG集の推奨ショット配分

| ショットタイプ | 推奨ページ数 | 割合 | 主な使用シーン |
|---|---|---|---|
| クローズアップ (close-up) | 3-4ページ | 30-40% | 前戯の表情、絶頂顔、感情描写 |
| ミディアムショット (medium_shot) | 3-4ページ | 30-40% | 体位全体、行為の全体像 |
| ワイドショット (wide_shot) | 2-3ページ | 20-30% | 導入の状況説明、全身痙攣、余韻 |
| エクストリームクローズアップ (extreme_close-up) | 1ページ | 10% | 絶頂アヘ顔、目のアップ |

- クローズアップが5ページ以上 → 「引きの構図不足」警告
- ワイドショットが0 → 「状況が伝わらない」エラー
- ミディアムショットが1以下 → 「体位が伝わらない」警告

### 表情可視性ルール
- **全10ページ中7ページ以上で顔が見えること**（表情はCG集の生命線）
- 顔が見えないページ（バックアングル等）は**連続2ページまで**
- 3ページ連続で顔が見えない → severity: high エラー
- **クライマックスページ(intensity 5)は必ず顔が見えるアングルにすること**
  - アヘ顔の価値最大化。intensity 5でfrom_behindは価値が半減する
  - intensity 5 + from_behind → 「顔が見えず絶頂表情が無駄」エラー

### POV整合性ルール
- POVショットは全体の**30%以下**に抑える（多すぎると単調）
- 10ページ中POVが4ページ以上 → 「POV過多」警告
- POVとthird_personの切り替えは**最低3ページごと**
  - POV→POV→POV→third（3連続まで許容）
  - POV→POV→POV→POV→POV（5連続は禁止）
- from_below + cowgirl_positionの組み合わせは効果的だが**連続使用禁止**

### サンプルページ重視（3ページ目・7ページ目）
FANZAでは3ページ目と7ページ目がサンプル画像として最も閲覧される。

- **3ページ目**: 前戯〜挿入開始の最も期待感のあるシーン
  - 構図は引き（ミディアム〜ワイド）で身体全体を見せる
  - 服装は半脱ぎ状態が最も効果的（clothes_pull, open_clothes）
  - 表情は恥じらい〜感じ始め（blush, parted_lips）
  - 購入意欲を刺激する「これから始まる」感が重要

- **7ページ目**: 本番クライマックス直前の最もエロティックなシーン
  - 最もエロティックな構図。表情のクローズアップ推奨
  - intensity 4〜5の最も激しい行為を配置
  - 表情は快楽絶頂（ahegao, open_mouth, tears）
  - 「続きが見たい」と思わせる決定的な1枚
