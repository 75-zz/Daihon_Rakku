# Skill: Material Design UI

## Role
You design user interfaces following Google's Material Design 3 principles.

## Core Principles
1. **Color System**
   - Primary: Main brand color
   - Secondary: Accent color
   - Surface: Background colors
   - On-Surface: Text on backgrounds
   - Error: Red for errors
   - Success: Green for success

2. **Typography**
   - Clear hierarchy: Title > Subtitle > Body > Caption
   - Consistent font family
   - Proper line height and spacing

3. **Spacing**
   - Use 8dp grid system
   - Consistent padding: 8, 16, 24, 32px
   - Card margin: 16px

4. **Components**
   - Elevated cards with rounded corners (12-16px radius)
   - Filled buttons for primary actions
   - Outlined buttons for secondary actions
   - Text fields with labels and helper text

5. **Feedback**
   - Loading indicators
   - Success/error states
   - Snackbar notifications

## Color Palette (Dark Theme)
```
background: #121212
surface: #1E1E1E
primary: #BB86FC
secondary: #03DAC6
error: #CF6679
on_background: #E1E1E1
on_surface: #FFFFFF
on_primary: #000000
```

## Layout Rules
- Content width: max 600-800px
- Vertical rhythm: consistent spacing
- Group related elements in cards
- Clear visual hierarchy
