## 絶対ルール（違反厳禁）

1. **♡禁止**: 男性セリフに♡♥を絶対に使わない。♡は女性キャラ専用
2. **甘え語尾禁止**: 「きもちぃ」「すきぃ」「もっとぉ」等の語尾伸ばし甘え表現は男性に不可
3. **否定甘え禁止**: 「やだ」「だめ」「いや…」等の否定甘え表現は男性に不可
4. **喘ぎ禁止**: 男性キャラにmoanタイプの吹き出しは原則禁止。「はぁん」「あっ」等の喘ぎ系も不可
5. **一人称**: 「俺」がデフォルト。紳士キャラのみ「僕」可。「私」「わたし」は男性に絶対不可
6. **丁寧語禁止**: 「です」「ます」「ください」は男性セリフに使わない（初対面の上司キャラのみ例外）
7. **最大8文字**: 男性セリフは女性より短く。男は多くを語らない
8. **1ページ0-1個**: 男性セリフは最低限。女性の喘ぎ・感情が主役

## テーマ別トーンマッピング

### NTR / 寝取り / 夜這い / 陵辱
- **基本口調**: 粗野・支配的・命令形
- **OK例**: 「おとなしくしろ」「すげえ締まる」「もう俺のだ」「鳴けよ」「逃がさねえ」
- **NG例**: 「気持ちいい？」「大丈夫？」「好きだよ」

### 純愛 / ラブラブ / 恋人
- **基本口調**: 優しいが男らしく。短文断定
- **OK例**: 「好きだよ」「気持ちいい？」「もっと見せて」「可愛いな」
- **NG例**: 「おとなしくしろ」「脱げ」「鳴けよ」

### 調教 / 堕落 / 催眠
- **基本口調**: 支配と褒めの混合。冷静な命令
- **OK例**: 「いい子だな」「もっと堕ちろ」「わかってるな」「よくできた」
- **NG例**: 「痛くない？」「無理しないで」

### 痴漢 / 盗撮
- **基本口調**: 小声の卑猥な囁き。短い断定
- **OK例**: 「ここ好きだろ」「声出せねえよな」「バレるぞ」「感じてる」
- **NG例**: 長文の説明セリフ

## 文法パターン

男性セリフの基本形は以下のいずれか:
- **命令形**: 「見ろ」「鳴け」「力抜け」
- **体言止め**: 「最高」「すげえ」「たまんねえ」
- **短い断定**: 「俺のだ」「もう戻れねえぞ」
- **挑発疑問**: 「どうした？」「まだいけるか？」（本当の質問ではなく挑発）
